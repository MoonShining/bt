# AI 炒股回测 & 通知项目

## 项目概述

基于 Backtrader 框架实现的股票趋势跟随策略回测系统，支持多维度技术指标分析和微信通知功能。

## 策略核心逻辑：多周期趋势跟随策略

**MultiPeriodTrendStrategy** - 基于 SMA、RSI、成交量的简单趋势跟随策略

核心理念：只在短、中、长期趋势一致向上时入场，逢低买入，分批止盈，固定止损。

### 1. 趋势判断：多周期均线多头排列

使用三层 SMA（简单移动平均线）判断趋势方向，要求**短中长期一致向上**才考虑入场：

- **短期均线**：SMA(10) - 判断短期趋势
- **中期均线**：SMA(30) - 判断中期趋势  
- **长期均线**：SMA(60) - 判断长期趋势

**趋势条件**：`SMA10 > SMA30 > SMA60`

当三条均线呈多头排列时，确认上涨趋势，避免在震荡或下跌趋势中交易。

### 2. 入场过滤：RSI 回调 + 放量确认

**RSI 条件**：
- RSI 周期：14
- 要求：`30 ≤ RSI ≤ 60`
- 目的：在上涨趋势中等待回调，避免追高，在相对低位入场

**成交量条件**：
- 成交量均线周期：20
- 要求：`当前成交量 > 1.2 × 成交量均线`
- 目的：放量确认上涨动能，验证趋势真实性

### 3. 入场条件（需同时满足）

| 条件 | 要求 |
|------|------|
| 趋势方向 | SMA5 > SMA20 > SMA60 |
| RSI 回调 | 30 ≤ RSI(14) ≤ 50 |
| 放量确认 | 成交量 > 1.5 × 成交量均线(20) |
| 仓位空间 | 未达到最大仓位限制 |

满足条件即可建仓，**支持多次加仓**（无论是否已有持仓），只要还有仓位空间且满足信号就可以继续加仓。

### 4. 风险管理

**固定止损**：
- 止损比例：10%（可配置）
- 止损价 = 平均建仓成本 × (1 - 止损比例)
- 每次加仓后重新计算止损价

**趋势反转止损**：
- 当 `SMA10 < SMA30`（短期均线跌破中期均线）立即清仓止损

**仓位管理：基于趋势强度的动态风险分配**

趋势越强，投入风险越大。综合三个维度计算趋势强度：

| 维度 | 计算方式 | 权重 |
|------|----------|------|
| 均线偏离度 | `(SMA10 - SMA60) / SMA60`，偏离越大越强 | 40% |
| RSI 位置 | RSI 越低（回调越深）强度越高 | 30% |
| 放量程度 | 成交量相对放大倍数越大越强 | 30% |

最终强度范围 `[0, 1]`，映射到风险比例：
- 最小风险：总资金 5%（弱趋势）
- 最大风险：总资金 20%（强趋势）

计算公式：
```
单次风险比例 = 最小风险 + 强度 × (最大风险 - 最小风险)
单次建仓数量 = 总资金 × 单次风险比例 ÷ (建仓价 × 止损比例)
```

- 最大仓位：总资金的 95%，超出后不再加仓
- 平均成本：多次建仓使用加权平均计算持仓成本

### 5. 分批止盈（金字塔式减仓）

盈利达到目标位后逐步卖出锁定利润：

| 止盈级别 | 盈利条件 | 卖出比例 |
|----------|----------|----------|
| 第一止盈 | 盈利 > 20% | 卖出 1/3 |
| 第二止盈 | 盈利 > 40% | 再卖出 1/3 |
| 第三止盈 | 盈利 > 60% **或**趋势不再向上 | 清仓剩余 |

分批止盈让利润奔跑，同时逐步锁定已有盈利。

### 6. 交易规则

- **方向**：仅做多，不做空
- **开仓**：支持多次建仓/加仓，满足信号即可买入
- **成本计算**：使用实际成交价计算加权平均持仓成本
- **止损**：固定比例止损 + 趋势反转止损
- **止盈**：分批止盈，盈利达标后逐步减仓

### 回测结果（1995-2014，ORCL，初始资金 100,000）

| 指标 | 结果 |
|------|------|
| 最终资金 | **132,080** |
| 总收益 | **+32.08%** |
| 年化收益 | **+1.40%** |
| 夏普比率 | 0.09 |
| 最大回撤 | **53.19%** |
| 最大回撤金额 | 132,330 |
| 最长回撤天数 | 3699 |
| 总交易次数 | 66 |
| 胜率 | 37.88% (25胜 / 41负) |

### 参数建议

- **保守型**：`base_risk=0.01, max_risk=0.04` → 收益 +61.1%，年化 +2.41%，回撤更小
- **激进型**：`base_risk=0.05, max_risk=0.20` → 当前参数，收益 +32.1%，年化 +1.40%，最大回撤 53%

风险放得过大（单次 5%-20%）会导致回撤显著增加，收益反而下降。
