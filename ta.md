# Backtrader TA-Lib 指标中文翻译

### 1. 数学变换 (Math Transform)
*   **ACOS**: Vector Trigonometric ACos - 向量反余弦函数
*   **ASIN**: Vector Trigonometric ASin - 向量反正弦函数
*   **ATAN**: Vector Trigonometric ATan - 向量反正切函数
*   **CEIL**: Vector Ceil - 向量向上取整
*   **COS**: Vector Trigonometric Cos - 向量余弦函数
*   **COSH**: Vector Trigonometric Cosh - 向量双曲余弦函数
*   **EXP**: Vector Arithmetic Exp - 向量指数函数
*   **FLOOR**: Vector Floor - 向量向下取整
*   **LN**: Vector Log Natural - 向量自然对数
*   **LOG10**: Vector Log10 - 向量以10为底的对数
*   **SIN**: Vector Trigonometric Sin - 向量正弦函数
*   **SINH**: Vector Trigonometric Sinh - 向量双曲正弦函数
*   **SQRT**: Vector Square Root - 向量平方根
*   **TAN**: Vector Trigonometric Tan - 向量正切函数
*   **TANH**: Vector Trigonometric Tanh - 向量双曲正切函数

### 2. 数学运算符 (Math Operators)
*   **ADD**: Vector Arithmetic Add - 向量加法
*   **DIV**: Vector Arithmetic Div - 向量除法
*   **MAX**: Highest value over a specified period - 指定周期内的最高值
*   **MAXINDEX**: Index of highest value over a specified period - 指定周期内最高值的索引
*   **MIN**: Lowest value over a specified period - 指定周期内的最低值
*   **MININDEX**: Index of lowest value over a specified period - 指定周期内最低值的索引
*   **MINMAX**: Lowest and highest values over a specified period - 指定周期内的最低和最高值
*   **MINMAXINDEX**: Indexes of lowest and highest values over a specified period - 指定周期内最低和最高值的索引
*   **MULT**: Vector Arithmetic Mult - 向量乘法
*   **SUB**: Vector Arithmetic Substraction - 向量减法
*   **SUM**: Summation - 求和

### 3. 价格变换 (Price Transform)
*   **AVGPRICE**: Average Price - 平均价格
*   **MEDPRICE**: Median Price - 中位数价格
*   **TYPPRICE**: Typical Price - 典型价格
*   **WCLPRICE**: Weighted Close Price - 加权收盘价

### 4. 波动率指标 (Volatility Indicators)
*   **ATR**: Average True Range - 平均真实波幅
*   **NATR**: Normalized Average True Range - 归一化平均真实波幅
*   **TRANGE**: True Range - 真实波幅

### 5. 成交量指标 (Volume Indicators)
*   **AD**: Chaikin A/D Line - 蔡金集散线 (Chaikin A/D Line)
*   **ADOSC**: Chaikin A/D Oscillator - 蔡金集散震荡指标 (Chaikin A/D Oscillator)
*   **OBV**: On Balance Volume - 能量潮指标 (On Balance Volume)

### 6. 周期指标 (Cycle Indicators)
*   **HT_DCPERIOD**: Hilbert Transform - Dominant Cycle Period - 希尔伯特变换：主导周期
*   **HT_DCPHASE**: Hilbert Transform - Dominant Cycle Phase - 希尔伯特变换：主导周期相位
*   **HT_PHASOR**: Hilbert Transform - Phasor Components - 希尔伯特变换：相量分量
*   **HT_SINE**: Hilbert Transform - SineWave - 希尔伯特变换：正弦波
*   **HT_TRENDMODE**: Hilbert Transform - Trend vs Cycle Mode - 希尔伯特变换：趋势与周期模式

### 10. 模式识别 / K线形态 (Pattern Recognition)

这里将 61 种 K 线形态按照构成形态所需的 K 线数量进行了分类，方便学习和记忆。

#### 10.1 单根 K 线形态 (Single-Candle Patterns)
这些形态仅由一根 K 线构成，通常反映了单日市场情绪的极端变化或犹豫不决。
*   **CDLBELTHOLD**: Belt-hold - 捉腰带线
*   **CDLCLOSINGMARUBOZU**: Closing Marubozu - 收盘光头光脚
*   **CDLDOJI**: Doji - 十字星
*   **CDLDRAGONFLYDOJI**: Dragonfly Doji - 蜻蜓十字星 (长下影十字星)
*   **CDLGRAVESTONEDOJI**: Gravestone Doji - 墓碑十字星 (长上影十字星)
*   **CDLHAMMER**: Hammer - 锤子线
*   **CDLHANGINGMAN**: Hanging Man - 上吊线
*   **CDLHIGHWAVE**: High-Wave Candle - 高浪线
*   **CDLINVERTEDHAMMER**: Inverted Hammer - 倒锤子线
*   **CDLLONGLEGGEDDOJI**: Long Legged Doji - 长腿十字星
*   **CDLLONGLINE**: Long Line Candle - 长实体K线
*   **CDLMARUBOZU**: Marubozu - 光头光脚
*   **CDLRICKSHAWMAN**: Rickshaw Man - 黄包车夫线
*   **CDLSHOOTINGSTAR**: Shooting Star - 流星线
*   **CDLSHORTLINE**: Short Line Candle - 短实体K线
*   **CDLSPINNINGTOP**: Spinning Top - 纺锤线
*   **CDLTAKURI**: Takuri (Dragonfly Doji with very long lower shadow) - 探水竿 (带长下影线的蜻蜓十字星)

#### 10.2 双根 K 线形态 (Two-Candle Patterns)
由两根 K 线组成，通常反映了市场情绪的快速反转或延续。
*   **CDLCOUNTERATTACK**: Counterattack - 反击线
*   **CDLDARKCLOUDCOVER**: Dark Cloud Cover - 乌云盖顶
*   **CDLDOJISTAR**: Doji Star - 十字星线
*   **CDLENGULFING**: Engulfing Pattern - 吞没形态 (抱线)
*   **CDLHARAMI**: Harami Pattern - 孕线形态
*   **CDLHARAMICROSS**: Harami Cross Pattern - 十字孕线形态
*   **CDLHOMINGPIGEON**: Homing Pigeon - 家鸽形态
*   **CDLINNECK**: In-Neck Pattern - 颈内线
*   **CDLKICKING**: Kicking - 踢水形态 (反击线)
*   **CDLKICKINGBYLENGTH**: Kicking - bull/bear determined by the longer marubozu - 踢水形态 (由较长的光头光脚线决定多空)
*   **CDLMATCHINGLOW**: Matching Low - 相同低点
*   **CDLONNECK**: On-Neck Pattern - 颈上线
*   **CDLPIERCING**: Piercing Pattern - 刺透形态
*   **CDLSEPARATINGLINES**: Separating Lines - 分离线
*   **CDLTHRUSTING**: Thrusting Pattern - 插入线

#### 10.3 三根 K 线形态 (Three-Candle Patterns)
由三根 K 线组成，能够提供比双根 K 线更强烈的反转或延续信号。
*   **CDL2CROWS**: Two Crows - 两只乌鸦
*   **CDL3BLACKCROWS**: Three Black Crows - 三只乌鸦
*   **CDL3INSIDE**: Three Inside Up/Down - 三内部上涨和下跌 (包含)
*   **CDL3OUTSIDE**: Three Outside Up/Down - 三外部上涨和下跌 (包容)
*   **CDL3STARSINSOUTH**: Three Stars In The South - 南方三星
*   **CDL3WHITESOLDIERS**: Three Advancing White Soldiers - 三个白武士 (红三兵)
*   **CDLABANDONEDBABY**: Abandoned Baby - 弃婴形态
*   **CDLADVANCEBLOCK**: Advance Block - 大敌当前
*   **CDLEVENINGDOJISTAR**: Evening Doji Star - 黄昏十字星
*   **CDLEVENINGSTAR**: Evening Star - 黄昏星
*   **CDLGAPSIDESIDEWHITE**: Up/Down-gap side-by-side white lines - 向上/下跳空并列阳线
*   **CDLIDENTICAL3CROWS**: Identical Three Crows - 相同三乌鸦
*   **CDLMORNINGDOJISTAR**: Morning Doji Star - 早晨十字星
*   **CDLMORNINGSTAR**: Morning Star - 早晨之星
*   **CDLSTALLEDPATTERN**: Stalled Pattern - 停顿形态
*   **CDLSTICKSANDWICH**: Stick Sandwich - 探底线 (条形三明治)
*   **CDLTASUKIGAP**: Tasuki Gap - 塔苏基跳空
*   **CDLTRISTAR**: Tristar Pattern - 三星形态
*   **CDLUNIQUE3RIVER**: Unique 3 River - 奇特三河床
*   **CDLUPSIDEGAP2CROWS**: Upside Gap Two Crows - 向上跳空两只乌鸦
*   **CDLXSIDEGAP3METHODS**: Upside/Downside Gap Three Methods - 向上/下跳空三法

#### 10.4 多根 K 线及复杂形态 (Multi-Candle / Complex Patterns)
由四根或五根以上的 K 线组成，反映了较长一段时间内市场多空力量的复杂博弈。
*   **CDL3LINESTRIKE**: Three-Line Strike - 三线打击
*   **CDLBREAKAWAY**: Breakaway - 脱离形态
*   **CDLCONCEALBABYSWALL**: Concealing Baby Swallow - 藏婴吞咽
*   **CDLHIKKAKE**: Hikkake Pattern - 陷阱形态
*   **CDLHIKKAKEMOD**: Modified Hikkake Pattern - 修正陷阱形态
*   **CDLLADDERBOTTOM**: Ladder Bottom - 阶梯底部
*   **CDLMATHOLD**: Mat Hold - 铺垫形态
*   **CDLRISEFALL3METHODS**: Rising/Falling Three Methods - 上升/下降三法

#### 10.5 常见强烈看涨形态 (Strong Bullish Patterns)
这些形态通常出现在下跌趋势的末端，释放出强烈的反转向上信号。
*   **CDL3WHITESOLDIERS**: Three Advancing White Soldiers - 三个白武士 (红三兵)
*   **CDLENGULFING**: Engulfing Pattern - 吞没形态 (抱线)
*   **CDLMORNINGSTAR**: Morning Star - 早晨之星
*   **CDLMORNINGDOJISTAR**: Morning Doji Star - 早晨十字星
*   **CDLPIERCING**: Piercing Pattern - 刺透形态
*   **CDLHAMMER**: Hammer - 锤子线
*   **CDL3INSIDE**: Three Inside Up/Down - 三内部上涨和下跌 (包含)
*   **CDL3OUTSIDE**: Three Outside Up/Down - 三外部上涨和下跌 (包容)
*   **CDLABANDONEDBABY**: Abandoned Baby - 弃婴形态
*   **CDLCONCEALBABYSWALL**: Concealing Baby Swallow - 藏婴吞咽
*   **CDLKICKING**: Kicking - 踢水形态 (反击线)
*   **CDLMATCHINGLOW**: Matching Low - 相同低点
*   **CDLMATHOLD**: Mat Hold - 铺垫形态
*   **CDLUNIQUE3RIVER**: Unique 3 River - 奇特三河床

#### 10.6 常见强烈看跌形态 (Strong Bearish Patterns)

*   **CDL3BLACKCROWS**: Three Black Crows - 三只乌鸦
*   **CDLDARKCLOUDCOVER**: Dark Cloud Cover - 乌云盖顶
*   **CDLEVENINGSTAR**: Evening Star - 黄昏星
*   **CDLEVENINGDOJISTAR**: Evening Doji Star - 黄昏十字星
*   **CDLENGULFING**: Engulfing Pattern - 吞没形态 (抱线)
*   **CDLSHOOTINGSTAR**: Shooting Star - 流星线

*   **CDL3INSIDE**: Three Inside Up/Down - 三内部上涨和下跌 (包含)
*   **CDL3OUTSIDE**: Three Outside Up/Down - 三外部上涨和下跌 (包容)
*   **CDLABANDONEDBABY**: Abandoned Baby - 弃婴形态
